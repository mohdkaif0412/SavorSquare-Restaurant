## Run Locally :-
To run SavorSquare locally, run this command on your git bash:

**Linux and macOS:**
``` bash  
sudo git clone https://github.com/Kaif0412/SavorSquare-Restaurant
```
**Windows:**
``` bash  
git clone https://github.com/Kaif0412/SavorSquare-Restaurant
```
